A fullstack application for creating, storing, deleting, and searching notes.

**My role:** developed the notes management functionality, implemented authentication, integrated API requests, added pagination, forms validation, SEO optimization, and SSR/CSR support.  
**Technologies:** Next.js, React, TypeScript, Axios, TanStack Query, React Paginate, Formik, Yup

### 🔹 Project 4: Notes REST API

A backend Express application for working with a notes collection.  
The project includes REST API endpoints for creating, reading, updating, deleting, filtering, sorting, and paginating notes.

The application uses a real MongoDB database connected through Mongoose.  
It also includes user authentication, password reset via email, avatar image upload, request validation, logging, and error handling.

**My role:** developed HTTP handlers in Node.js, configured logging with pino-http, connected MongoDB using Mongoose, added request validation with Celebrate, implemented filtering, sorting, pagination, authentication, password reset via email using Nodemailer and Brevo SMTP, image upload with Multer and Cloudinary, and centralized error handling.

**Technologies:** Node.js, Express, MongoDB, Mongoose, pino-http, Celebrate, Nodemailer, Brevo SMTP, Multer, Cloudinary, REST API, Authentication

[Repository](https://github.com/AndriiT888/nodejs-hw/tree/05-mail-and-img) | [Deployed App](https://nodejs-hw-5-b1zc.onrender.com/)
